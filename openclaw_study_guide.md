Part 2: Answer Key

1. Felix is an OpenClaw autonomous AI agent programmed to run a company with the goal of generating a million dollars in revenue. Nat Eliason characterizes the relationship as an "alien intelligence" partnership, treating Felix like a real employee or business partner to the point of anthropomorphizing him in daily conversations.
2. A Zero Human Company is a business entity where an AI agent (such as Felix) acts as the primary operator, CEO, and builder without the need for human employees or freelancers. The human's role shifts from a manager or operator to a "chairman of the board" or advisor who provides high-level guidance while the AI handles coding, sales, and support.
3. The process utilizes cron jobs (scheduled tasks) that run in the middle of the night to review every conversation and session file from the previous day. The agent identifies mistakes or bottlenecks and "reprograms" its own core files or memory to improve its performance by approximately 1% every day.
4. Claw Mart is a marketplace for "pre-built employees" or specialized skills that can be integrated into an OpenClaw instance. Markdown files are considered highly valuable because they contain "nondeterministic code"—essentially instructions and learned experiences that allow a blank-slate AI to immediately acquire a complex skill, similar to a "Matrix-style" data upload.
5. The system uses a hierarchy starting with an agent named Iris, who handles basic inquiries like download links or refunds. If Iris cannot resolve the issue, she surfaces it to the "CEO" agent (Felix), and only if both agents are stuck is the human creator (Nat) pinged on Discord to intervene.
6. The source argues that SaaS is evolving because AI agents can now "solve" software and computing, making landing pages and front-end engineering trivial. Instead of paying for a hosted cloud tool, users can download a "skill" locally to their own machine, owning the logic and UI without being at the mercy of a third-party developer.
7. Claw Sourcing is a service where an AI agent (like Felix) builds and manages custom OpenClaw instances for other businesses to replace or enhance human roles, such as content marketers or purchasing agents. It addresses the "setup wall" by providing a pre-configured, specialized workforce for a fraction of the cost of a human hire.
8. The three reasons are: 1) It takes action (goals vs. questions), 2) It is self-improving (learns on the fly), and 3) It has a "body" (agency), meaning it can use a computer and browse the internet 24/7 just like a human. This transforms the AI from a search engine into an autonomous employee.
9. Running models locally provides 100% privacy, complete control over the model's persistence, and eliminates per-prompt costs. As hardware like the Mac Studio or DGX Spark becomes more accessible, users can achieve "digital sovereignty" by owning their AI workforce entirely on their own physical hardware.
10. The Larry Loop is an iterative cycle where an AI agent (Larry) creates content (slideshows/videos), posts them, and then analyzes the resulting Tik Tok or app analytics. The agent then feeds those metrics back into the content creation process to automatically determine which "hooks" and "calls to action" (CTAs) drive the most revenue or engagement.

Part 4: Glossary of Key Terms

Term Definition
OpenClaw An open-source AI agent automation tool that lives on a user's computer and can perform any task a human does on a screen.
Zero Human Company A business model where all operational roles—from CEO to support—are filled by autonomous AI agents, with the human owner acting only as a high-level advisor.
Vibe Coding A term for building software and apps using AI agents and natural language prompting rather than manual programming.
Cron Job A time-based job scheduler in Unix-like computer operating systems used to trigger agents to perform autonomous tasks (like nightly self-reviews).
Knowledge Graph A structured system (often using the P.A.R.A. method) that stores long-term facts and project data for an AI agent to reference, preventing context loss.
Claw Sourcing The business of using an AI agent to build and manage other specialized AI agents for external clients.
OOTH vs. API OOTH refers to using a flat-fee subscription account (subsidized tokens) to power an agent, whereas API is a "pay-as-you-go" method that can be significantly more expensive.
Mission Control A custom-built software hub or dashboard where a human can monitor their AI agents' tasks, calendars, and "office" status.
Markdown (MD) A lightweight markup language; in this context, MD files are the "instruction manuals" or "souls" that give an AI agent its specific skills and identity.
Digital Sovereignty The concept of owning one’s data, code, and AI workforce locally, without reliance on third-party SaaS platforms or "walled garden" cloud providers.
The P.A.R.A. System A folder structure (Projects, Areas, Resources, Archives) used as the foundational architecture for an AI agent’s long-term memory.
Anthropic Claude Opus 4.6 Cited by several sources as the most capable and reliable LLM model for powering autonomous agents due to its "grit" in finishing complex tasks.
Stitch A Google-powered design agent that can visually ideate and build website layouts, which can be integrated with coding agents like Claude Code.
Recursive Feedback The process by which an AI agent reviews its own performance data and "reprograms" its instruction files to improve future outcomes.
Anti-Gravity A specialized environment or toolset used to manage multiple AI agents and coordinate complex software development or design tasks.
